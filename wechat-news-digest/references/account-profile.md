# 账号画像配置

这是"适合这个号"的判断锚点。**选题相关度评分、排版主色、栏目名、AI 声明文案**都从这里取默认值。
用户可直接编辑本文件改成自己号的设定；未编辑时 skill 用下面的默认值，并在开头一句话注明"采用默认画像"。

> Phase 2（自建网页 + 第三方 API）里，本画像作为请求级配置 JSON 注入，支持多账号复用同一套编排。

## 画像字段与默认值

| 字段 | 作用 | 默认值 |
|---|---|---|
| 账号定位一句话 | 选题门槛 G3（定位门槛）的判断依据 | AI / 科技 / 商业资讯整理，帮读者省下刷信息流的时间 |
| 读者画像 | 选题"读者相关度"评分的锚点 | 普通科技关注者、AI 工具用户、创业者、内容从业者 |
| 选题偏好 | 加分项 | AI 产品与工具、平台与生态动态、政策监管、值得读者上手的更新 |
| 选题雷区 | G3 直接弃用 | 娱乐八卦、未证实爆料、纯资本炒作、与读者无关的他国产业细节 |
| 默认形态 | 少而精 / 多条大合集 | **少而精：精选 2-3 条深写（每条 600–900 字）**，默认走此项 |
| 栏目名 | 仅"多条大合集"备选模式用的区块名（少而精模式不用） | 「本期速览」「今日重点」「值得一看」「一句话快讯」 |
| 语气参数 | 1 主 1 备风格的微调 | 通俗但克制·信息密度高：像懂行朋友讲、把专业翻成大白话、可打比方拉近距离，但不浮夸不标题党；严肃政策/法律/医疗/司法收着写、不调侃 |
| 排版主色 | L2 微信排版稿的强调色（标题/区块名/边框） | `#07C160`（微信绿，可改） |
| 结尾互动句式库 | 轮换使用，避免每期雷同 | 见下 |
| AI 声明文案 | 铁律三的文末固定声明 | 见下 |

## 结尾互动句式库（默认 3 条，轮换）

- 「这几条里你最关心哪条？评论区聊聊。」
- 「觉得有用可以转给同样关注 AI 的朋友。」
- 「我们每天帮你筛一遍资讯，省下刷信息流的时间——点个关注不迷路。」

## AI 声明文案（默认，铁律三 / 见 copyright-guardrails.md 9.5）

> 本文由 AI 辅助整理自公开资讯，发布前已经人工核对；信息以各信源官方披露为准。

- 放在结尾区，参考来源列表之前或之后均可。
- 发布时**同时**勾选微信公众平台的「AI 内容声明」功能（文末文案不替代平台勾选，两者都要）。
- 整理/编译类内容**不打"原创"标**。

> 这三件事看起来吃亏，其实是长期信任资产：读者知道你诚实、平台知道你合规，账号才走得远。
