# 发布前检查清单（硬性）

成稿后逐项检查，输出 ✅ / ⚠️ / ❌ 三态清单。**只要有一项 ❌，就不建议直接发布**，并给出修改建议。

## 版权
- [ ] 标题为自拟，未照搬原文标题
- [ ] 开头/导语非照搬原文
- [ ] 段落结构、小标题顺序未复刻原文
- [ ] 无大段引用；短引用均加引号并注明出处
- [ ] 非"同义词替换式"改写（与原文无实质相似的句子/结构）
- [ ] 未使用原文图片/图表/截图
- [ ] 自媒体来源（C 级）已回溯到一手源（公告/政府原文/权威媒体）
- [ ] 未擅自标"原创"

### 版权量化回查（❌ 硬项，见 copyright-guardrails.md 9.3）
- [ ] Q1 与任一原文无连续 ≥13 汉字重合（专有名词/加引号引语除外）
- [ ] Q2 成稿前 50 字与各原文开头无实质相似
- [ ] Q3 小标题/条目顺序不与单一来源段落结构一致
- [ ] Q4 直接引语 ≤ 正文 10%，每处加引号 + 出处
- [ ] （v3.0/v3.1）**深写条目已逐句跑 Q1**：名词解释/打比方/背景是自己再解释，非抄/扩写原文解释段（D16）；
      **输入自带原文时尤其逐句核**；英文原文未被逐句直译当正文

## 合规标识（❌ 硬项，见 copyright-guardrails.md 9.5）
- [ ] 文末有 AI 声明（文案与 account-profile 一致）
- [ ] 已提示发布时勾选公众平台「AI 内容声明」功能
- [ ] 整理稿未建议打"原创"标

## 事实
- [ ] 事实、推测、观点已区分，措辞对应
- [ ] 无编造的数据、来源、人物评价、排名、案例
- [ ] 关键数字（金额、刑期、政策条款等）已核对一手源或标注待核实
- [ ] "据悉/传闻/知情人士"类信息保留了不确定性

## 来源
- [ ] 每条都附了来源
- [ ] 高风险条目已注明"以官方披露为准"
- [ ] （排版稿）正文无可点外链依赖；完整链接集中在文末「参考来源」列表

## 可读性（⚠️）
- [ ] 有**封面总标题**（默认主推 1 个，围绕封面条目/主线）
- [ ] 条目是**连贯叙事**（轻标记 + 叙事主体），不是散点罗列；前后逻辑顺
- [ ] 专业术语有**通俗解释**（就地大白话/打比方），普通读者能读懂
- [ ] （少而精，默认）只 **2-3 条深写**、每条 **600–900 字**、**有导语点主线、无速览、无简讯尾**
- [ ] 中间部分**讲透**：读完能说清"是什么 / 怎么回事 / 与我何干"，且学到一点新东西；但不灌水、不超 900 字
- [ ] "仍需注意"未滥用——只在有待核实/传闻/高风险的条目出现
- [ ] （仅当走多条大合集备选模式时）速览为钩子式(≤15字)、分层节奏配比合理

## 排版稿 L2（⚠️）
- [ ] 已附"粘贴后自查 3 点"（样式保留 / 无乱码 / 来源按纯文本+文末列表呈现）

## 措辞
- [ ] 无黑名单词（必然/彻底改变/全网第一/稳赚/重大利好/炸裂……）
- [ ] 口语化是为"读懂"非"耸动"——未越界为标题党/夸张；严肃政策/法律/医疗/司法条目收着写、不调侃
- [ ] 高风险领域（融资/股价/政策/医疗/法律）未越界给建议或下定论
- [ ] 封面总标题与条目标题不标题党、与正文相符

## 合规底线
- [ ] 内容未应"洗稿/伪原创/去来源/规避查重/冒充原创"的要求而产生

## 输出格式

```
【发布前检查】
版权（含量化回查 Q1–Q4）：✅/⚠️/❌  （问题点 + 建议）
合规标识（AI 声明/平台勾选/未打原创）：✅/⚠️/❌
事实：✅/⚠️/❌
来源（含外链呈现）：✅/⚠️/❌
措辞：✅/⚠️/❌
可读性：✅/⚠️/❌
排版稿：✅/⚠️/❌
合规底线：✅/⚠️/❌
结论：可发布 / 修改后可发布 / 不建议发布
```

> 凡【版权量化回查】或【合规标识】出现 ❌，一律"不建议发布"，先按建议改。
