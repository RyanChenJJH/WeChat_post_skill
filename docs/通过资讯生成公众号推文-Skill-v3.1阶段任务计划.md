# wechat-news-digest Skill v3.1 阶段任务计划

> 依据 `...-v3.1.md`。本计划负责"改哪个文件、写什么、顺序、验收"。`- [ ]` 跟踪。

**目标：** 少而精（默认 2-3 条深写、去速览/简讯）+ 输入带原文（联网改可选）+ 每条 600-900 字中间讲透，
**版权不可侵犯为前提**（原文在手→逐句 Q1 + D16 再解释）。

## 全局约束

- 默认 2-3 条深写；输入已预选 2-3 则全做不再筛。无简讯/无其他尾。
- 每条 600-900 字，中间叙事主体讲透机制/背景/影响/名词；导语点主线（不用速览）。
- 封面用主推总标题。
- 联网可选：优先用输入 `> [!quote]- 原文`；无原文且需核实才联网，不可用则降级。
- 版权：读原文→记事实笔记→不照搬→凭笔记重写；深写逐句 Q1（无连续≥13字重合）；英文原文不直译。
