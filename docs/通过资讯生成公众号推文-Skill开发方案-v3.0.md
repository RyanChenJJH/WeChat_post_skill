## 三、核心设计：正文条目 = 轻标记 + 连贯叙事（D12 + D13）

### 3.1 新条目骨架

```
[条目小标题]

发生了什么：<1 句硬事实，点明 主体/动作/时间，这是唯一仍"分点感"的轻标签>

<连贯叙事主体（头条 2–3 段 / 标准区 1–2 段）：
 · 关键信息（数字/模式/范围）织进句子，不再罗列成 bullet
 · 遇术语就地用大白话解释（原"名词卡"内容融进来）
 · 需要时补一句可核实的背景或时间线（原"背景卡"融进来）
 · 可打比方、举身边例子，把专业翻成大众能懂
 · 通俗但克制：说人话，不喊"颠覆/炸裂/必将"
 · 版权红线：这段的"丰富"来自你自己的再解释与多源综合，
   不是把原文那段扩写——见第五章 D16>

为什么值得关注：<1–2 句，对读者/行业的轻度意义>
（仍需注意：<条件字段，仅在有待核实/传闻/高风险时出现>）
来源：<媒体名 · 日期>
```

关键变化：**"关键信息""名词卡""背景卡"三个 v2.1 的独立小块，全部融进叙事主体**——
能力不删（通俗解释、补背景仍在），只是从"卡片堆叠"变成"行文里就地讲清楚"。

### 3.2 三区的叙事密度（节奏不变，写法变）

- **头条区（1–2 条）**：叙事主体 2–3 段，深写，名词/背景解释充分，可打比方。
- **标准区（3–4 条）**：叙事主体 1–2 段，精简但仍连贯成文，不退回罗列。
- **简讯区（2–3 条）**：保持一句话（含降级条目）。

### 3.3 通俗但克制的边界（D13）

- 口语**白名单**（鼓励）：说白了 / 打个比方 / 换句话说 / 这意味着 / 你可以理解为 / 憋的一步棋。
- **黑名单不变**：颠覆 / 必将 / 全网第一 / 稳赚 / 重大利好 / 炸裂 / 不容错过。
- 严肃政策/法律/医疗/司法条目：仍收着写，通俗解释可以，但不调侃、不轻佻。
- 一句话自检：通俗是为了"读懂"，不是为了"耸动"。

## 四、本期速览 + 封面总标题（D14 + D15）

### 4.1 本期速览 = 标题钩子式

每条 ≤15 字，一句戳中核心，编号列表，扫一眼就知道这期有啥、值不值得读。

```
📮 本期速览
1. 微信小程序，要能被 AI 一句话调用了
2. 月之暗面估值破百亿，资本还在追大模型
3. 网络测评新规：不实测，不能乱说
4. 国安部点名"AI 中转站"：当心数据外流
5. 华为讲清了：手机变卡是内存不是存储
```

### 4.2 封面总标题（整篇大标题）

- 给 **1 个主推 + 3 个备选**，围绕封面条目或当期主线。
- 区别于条目小标题：这是公众号文章的大标题（决定打开率），条目小标题是合集内每条的标题。
- 仍守标题安全：不标题党、不绝对化、不照搬原标题；可带「｜本期 AI 速递」后缀。
- 示例：
  - 主推：微信要让 AI 替你用小程序了｜本期 AI 速递
  - 备选：① AI 正在钻进微信、手机和监管 ② 这期 7 条，国产 AI 在悄悄加速 ③ 6/9 AI 速递：微信接入 AI、月之暗面破百亿

## 五、版权：丰富叙事下的核心原则（D16，最重要）

> 这是 v3.0 风险最高处。更丰富、更连贯的叙事，天然诱使作者"顺着原文那篇展开"——那就是洗稿。

### 5.1 一句话原则

**"涨知识"的增量来自 AI 自己的再解释，不是把原文那篇展开复述得更丰满。**

- ✅ 安全且增值：用通用常识把术语翻成大白话、多源综合、补可核实背景、打比方——
  这些是模型自身知识与组织，天然不撞任何单一原文的表达。
- ❌ 危险：把那篇报道的论述"扩写得更细更顺"——读起来丰富，本质是洗稿。

### 5.2 操作纪律（在 v2.1 反洗稿铁律上加一层）

1. 读源链接是为了**核实硬事实细节**（点 4 的要求）；读完**关掉**，凭笔记 + 自身常识写。
2. 名词解释、打比方、背景，优先用**自己的知识**写，不抄原文的解释段。
3. 叙事越丰富，**量化回查（连续 ≥13 字重合）越要逐句跑**——丰富不能靠贴原文换来。
4. 多源综合：同一事尽量 ≥2 源交叉，用自己的结构讲，天然远离单一原文。

> 落地：写进 `copyright-guardrails.md` 作为"事实 vs 表达"原理的深化与铁律一补充；
> `prepublish-checklist.md` 对"丰富叙事条目"强调重点跑 Q1。
