## 三、核心变化：少而精 + 深写（D18 + D19）

### 3.1 新的整体结构（≤3 条模式，默认）

条数少，**取消分层节奏与速览**，结构拉直：

```
[封面总标题（主推）]

[导语：1-2 句点出本期主线/为什么这几条值得花时间]

[条目 1：深写 600-900 字]
[条目 2：深写 600-900 字]
([条目 3：深写 600-900 字])

[结尾区：克制收束 + 互动引导 + AI 声明 + 参考来源]
```

> 注：分层节奏版（速览/头条/标准/简讯）保留为**备选**，仅当用户明确要"多条大合集"时用。默认走少而精。

### 3.2 单条深写骨架（D19，中间部分是重点）

```
[条目小标题]

发生了什么：<1-2 句硬事实，主体/动作/时间>

<连贯叙事主体（这是深写的核心，约 400-700 字，分 2-4 个自然段）：
 把"中间部分"讲透——按需覆盖：
 · 来龙去脉：这事怎么发生的、此前的背景/时间线
 · 机制/原理：它到底怎么做到的（用大白话 + 打比方，原"名词卡"在此展开）
 · 关键细节与数字：织进句子讲，不罗列（数字以原文/官方为准）
 · 影响/意义：对读者、对行业意味着什么（轻度，克制）
 · 张力/争议：如有不同声音、限制、不确定，如实带出
 全程通俗但克制；版权见 3.3>

为什么值得关注：<1-2 句收束本条对读者的意义>
（仍需注意：<条件字段，待核实/传闻/高风险时>）
来源：<媒体名 · 日期>
```

中间主体写"够"的判断：读者读完能**说清这事是什么、怎么回事、与我何干**，且学到一点新东西。

### 3.3 深写与版权（R6，最高优先，原文在手时尤其要守）

**原文就在输入里 → 抄起来更顺手 → 洗稿风险反而更高。** 因此 v3.1 的深写必须更严：

1. **铁律照旧**：读输入里的原文 → 提炼硬事实记笔记 → **不照着原文段落改写**，凭笔记 + 自身常识，用自己的结构和话重写。
2. **D16 再解释**：600-900 字的"丰富"靠你**自己把事情讲明白**（背景、打比方、机制解释来自你的知识），不是把原文那几段扩写/翻译。
3. **逐句量化回查（强制）**：深写每段成稿后，对照输入原文逐句跑 Q1——**与原文无连续 ≥13 汉字重合**（专有名词、加引号直接引语除外）；命中即重写。字越多越要逐句查。
4. **直接引语**：确需引用原文一两句，加引号 + "据 XX"，且占比极小（≤正文 10%）。
5. **英文原文**：翻译≠原创，更不能逐句直译当正文；同样要从事实重写、用自己的话讲。

## 四、输入与流程调整（D17）

### 4.1 输入类型（更新）

1. **带原文的资讯 md（新主力）**：每条含摘要 + 背景 + 参考链接 + `> [!quote]- 原文` 全文块。**直接用原文块提取事实，无需联网。**
2. 预选的 2-3 条（带原文）：直接深写，不再筛选。
3. 旧式抓取卡片（仅摘要+链接，无原文）：联网可用时读链接；不可用时按断网降级（仅用摘要硬事实、未证实标待核实）。
4. 用户直接粘贴的文本/链接：同上。

### 4.2 联网定位（重要变化）

- 联网从 v2.1/D2 的"**要求**联网读原文"改为"**可选**"：有原文就用原文；仅当输入无原文且需要核实时才联网，联网不可用则降级。
- 好处：① 主力场景（带原文）稳定不依赖网络；② 网页/第三方 API 端（本就难联网）天然适配。

## 五、最终输出结构（≤3 条模式）

```
【1. 选题与组稿说明】       ← 选了哪 2-3 条、为什么、主线（输入已预选则说明"按输入条目全做"）
【2. 封面总标题】           ← 主推 1 个（可附 2-3 备选）
【3. 公众号正文(L1)】       ← 导语点主线 + 2-3 条深写（每条 600-900 字）
【4. 微信排版稿(L2)】       ← 确定性转换，内联样式
【5. 来源与风险提示】       ← 以官方/原文为准、医疗等高风险降级
【6. 版权与合规检查】       ← 逐句 Q1 结果 + D16 自查 + AI 声明 + 未打原创
【7. 配图建议(文字版)】
【8. 变现建议】
```

## 八、风险与缓解

| 风险 | 缓解 |
|---|---|
| 原文在手 → 深写更易洗稿（v3.1 最大风险）| 逐句 Q1 强制 + D16 再解释 + 英文不直译；checklist 硬项 |
| 深写 → 掺入推测当事实 / 医疗越界 | 事实/推测/观点三分照旧；医疗只整理不下诊疗结论；数字以原文官方为准 |
| 2-3 条 → 漏掉当期其他重点 | 用户已确认接受（要质不要量）；选题说明里可一句话提"另有 N 条未选" |
| 深写 → 过长啰嗦 | 600-900 字上限 + "读完能说清三点"自检；不灌水 |
