# wechat-news-digest Skill v3.0 撰写阶段实施计划

> **给执行者：** 把 v3.0 方案（同目录 `...-v3.0.md`）的第七章实施清单拆成可执行撰写任务。
> 内容规范以 v3.0 方案为准；本计划负责"改哪个文件、写什么、顺序、怎么验收"。`- [ ]` 跟踪。

**目标：** 把 `wechat-news-digest/` 从 v2.1 升级到 v3.0——正文条目改"轻标记+连贯叙事"、语气通俗但克制、
速览钩子式、新增封面总标题、版权加 D16 原则。**不推翻 v2.1 框架。**

## 全局约束（每个任务隐含）

- 条目骨架：`发生了什么`(轻标记一句) → 连贯叙事主体(融关键信息+通俗解释+背景+打比方) → `为什么值得关注` → `仍需注意`(条件) → `来源`。
- 名词卡/背景卡：**融入叙事**，不再独立小块（能力保留，呈现方式变）。
- 语气：通俗但克制；口语白名单(说白了/打个比方)可用，黑名单(颠覆/必将/稳赚)不变。
- 版权 D16：丰富来自 AI 再解释，非展开复述原文；叙事越丰富越要逐句跑 Q1(13 字)。
- 封面总标题：1 主 + 3 备选，区别于条目小标题。
